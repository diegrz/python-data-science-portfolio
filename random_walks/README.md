# 2D Random Walk Simulation ⚛️

## Project Overview
This repository contains a Python implementation of a **2D Random Walk**, a fundamental stochastic process used to model the chaotic, unpredictable paths of particles (conceptually similar to Brownian motion and molecular diffusion). 

The project focuses on generating pseudo-random coordinates and utilizing advanced data visualization techniques to map the temporal and spatial evolution of the particle's trajectory.

---

## Visualizations

### 1. High-Density Trajectory (Static Mapping)
The static visualization simulates a random walk consisting of **50,000 discrete steps**. 
* **Temporal Encoding:** A cyan color gradient maps the time evolution of the walk. Lighter shades represent early stages, while darker shades indicate later stages.
* **Displacement Markers:** The starting position is anchored by a green dot, and the final displacement is marked by a red dot.

![High-Density Random Walk](rw_evidencia.jpg)

### 2. Real-Time Animation
To better illustrate the stochastic nature and the continuous generation of the path, the trajectory is also rendered as a time-lapse animation. This visualizes the step-by-step geometric progression of the particle in real-time.

![Random Walk Animation](nombre_de_tu_animacion.gif)

> *Note: The animation demonstrates the frame-by-frame coordinate updates generated by the algorithm.*

## Theoretical Framework
A Random Walk describes a path consisting of a succession of random steps. In this 2D discrete system, the position of the particle at any step $i+1$ is determined by:

$$x_{i+1} = x_i + \Delta x$$
$$y_{i+1} = y_i + \Delta y$$

Where $\Delta x$ and $\Delta y$ are independent random variables dictating the direction and distance of the next step.
